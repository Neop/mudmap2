MUD Map Keyboard Shortcuts

You can use these keyboard shortcuts to speed up your work flow.

Be aware that there is no undo in MUD Map, so remember to save your world often!

World File

Key combinationEffect
ctrl + nCreate new world
ctrl + sSave world
ctrl + alt + sSave world as...
ctrl + oShow open world dialog
ctrl + eExport map as image
ctrl + fSearch for map or place (see side bar)

Map navigation

You can create your world by using your pointing device or by using the selection cursor which can be controlled by the direction keys on your keyboard. The selection cursor is displayed on your map as a red box.

Note: direction keys are the arrow keys, w/a/s/d and the numpad numbers (except 5).

Key combinationEffect
pEnable / disable place selection cursor
right clickShow context menu of the map tile pointed at
context menu keyShow context menu of selected place
+/- and page up/downZoom in and out
h / home / numpad 5Go to home position
direction keysMove cursor
ctrl + fFocus search box in side bar

Map manipulation

Place creation

Key combinationEffect
e / enter / insert / double clickCreate / edit place at selection cursor
fCreate a placeholder
r / deleteRemove selected place
cEdit place comments
qModify group of selected place. Note that this affects all places that this group has been assigned to. A new group will be created if no place is selected.

Path creation

Key combinationEffect
ctrl + numpad 5Show create path dialog
ctrl + direction keysCreate path to adjacent place (no w/a/s/d). If there is no adjacent place or if its exit is occupied a 'exit to unknown target'-marker will be created.
alt + direction keysRemove all plaths to adjacent place

Place selection

You can select several places at once to change a property to a common value or to move / copy them. Selected places are displayed in pale colors. If you cut or copied places blue place cursors will show you where each place will be inserted.

Warning! Copy / paste operations can not be undone! Make sure to save your world before using this!

Key combinationEffect
ctrl + clickAdd / remove place from selection
spaceAdd / remove place marked by selection
shift + clickBox selection
shift + direction keysBox selection
ctrl + aSelect all places on current map
escapeDeselect everything
ctrl + xCut selected places
ctrl + xCopy selected places
Ctrl + vPaste places (this can not be undone!)
